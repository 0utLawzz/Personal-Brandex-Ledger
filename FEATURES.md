# Features

- Enterprise-grade Document Management System for IP law practices.
- Client, Case, and Payment management with full CRUD operations.
- Client Ledger for financial tracking in PKR, including rolling balances and Recharts visualization.
- OpenAPI specification as the single source of truth for backend endpoints and frontend hooks (via Orval).
- Professional, dense UI design (react-resizable-panels) tailored for enterprise workflows.
